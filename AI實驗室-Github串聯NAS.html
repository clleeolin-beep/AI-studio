混合雲協作邏輯：開發與數據的分離與整合

GitHub 部署端

前端介面維護： 代碼推送到 GitHub 後，透過 Actions 自動部署至 NAS Web Station。

● 版本控制 ● 自動化部署 ● 程式碼審核

NAS 數據中臺

數據資產本地化： 存放物料、圖資與核心 MariaDB 資料庫。提供 Python 高頻運算環境。

● 數據私有 ● 高速存取 ● 24/7 運算

Google Sheets 協作

低門檻數據輸入： 技術長授權雲端表單，同仁於熟悉界面更新資訊，Python 定期同步回 NAS。

● 零門檻協作 ● 權限控管 ● 異地備份

運維安全與治理 (CTO 指導)

權限管理

  • ● GitHub Secrets: 連線資訊加密，代碼庫不留憑證。
  • ● Google Service Account: 僅授權腳本讀取特定表單。
  • ● NAS Firewall: 嚴格限制僅公司內網或 VPN 可存取。

穩定性維護

  • ● 3-2-1 備援: 定期將 NAS 數據快照備份至異地。
  • ● 日誌審計: 所有爬蟲與部署行為皆有詳細日誌。
  • ● 故障告警: 腳本執行失敗即時發送 Line 通知開發者。
